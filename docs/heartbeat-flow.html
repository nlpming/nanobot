⏱ Timer — 每 interval_s 秒
读取 HEARTBEAT.md
Phase 1

System Prompt

User Message

Review the following HEARTBEAT.md and decide whether there are active tasks.

<HEARTBEAT.md 全文>
唯一工具 — heartbeat
// LLM 必须调用此工具，否则默认 skip
{
  "action": "skip" | "run",           // skip = 本轮跳过
  "tasks":  "需要执行的任务描述"   // run 时必填，传入 Phase 2
}
?
skip 无任务 / 文件为空 → 结束本轮，等待下次触发
run 有任务 → 携带 tasks 描述进入 Phase 2
action = "run"，传递 tasks 描述
Phase 2
执行 — 完整 Agent Loop
process_direct(tasks, session_key="heartbeat")
System Prompt — 与普通对话完全相同

read_file write_file edit_file list_dir exec web_search web_fetch message cron spawn MCP tools
🔒 session_key = "heartbeat"，独立于用户对话 session，保留最近 keep_recent_messages 条上下文
